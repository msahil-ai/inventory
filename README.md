A lightweight, fast, and interactive Command Line Interface (CLI) Inventory Management System built entirely in Python. This project is designed to provide seamless terminal-based management for tracking items, updating stock, and generating inventory summaries without the overhead of complex GUI applications.

## ✨ Features

* **➕ Add Items:** Easily register new items into your inventory.
* **🗑️ Delete Items:** Remove discontinued or out-of-stock items cleanly.
* **🔄 Update Stock:** Increment or decrement stock values on the fly.
* **🔍 Search:** Quickly look up specific items to check their details and availability.
* **📋 View Inventory:** Display your entire inventory in an organized format.
* **📊 Summaries:** Generate text-based summaries and counts (saves to `summary.txt` and `count_value.txt`).
* **💾 Persistent Storage:** Ensures data is stored safely through built-in storage handlers.
